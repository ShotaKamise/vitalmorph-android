# VitaMorph 71体進化表(v0.10)

2026-07-12ユーザー承認。全71体が入手可能になる構成で、旧43枠構成(`docs/MONSTER_ROSTER.md`)を置き換える。
キャラクターのID・名前・外見仕様は一切変更していない(ID規約は `docs/MONSTER_ASSET_CONTRACT.md` のまま)。

## 構成

```text
共通(性別なし・♂/♀マーク表示)   幼生1 + 成長体6          = 7
動物系(性別割り振り済み)         成熟体12 + 最終形態24     = 36
人型(性別別ID)                   成熟体14 + 最終形態14     = 28
合計                                                        71
```

## 孵化時に決まるもの

- **性別**(♂/♀ 約50%)
- **ルート適性**(人型/動物 約50%)— v0.10で追加。同じ生活でも世代ごとに違う進化を楽しめる

どちらも同じ世代では変わらず、能力差はない。v0.10より前から進行中の世代は
人型ルート固定として移行する(姿が突然変わらないようにするため)。

## 進化の流れ

```text
第1週  モルフィ(共通)
第2週  生活傾向で系統が決まり、成長体6種のどれかへ(共通)
第3週  ルート適性で分岐
        人型ルート: 第2週の傾向 → 7職業の成熟体(性別別)
        動物ルート: 系統×性別 → 動物成熟体
第4週  人型ルート: 生活・機嫌・絆 → 上位人型 or 動物系最終形態6体
        動物ルート: 第3週の生活 → 成熟体ごとの2候補から1体
```

## 動物系の性別割り振り

各系統の成熟体2体を♂/♀へ1体ずつ割り振り、最終形態は親の性別を継承する。
実装は `EvolutionEngine.animalFormSex`。

| 系統 | ♂成熟体 | ♂最終形態 | ♀成熟体 | ♀最終形態 |
|---|---|---|---|---|
| 調和 | ソルフェオン | アステリオン / クロノレオス | アクアネル | ミラフローラ / セレナディア |
| 筋力 | グランガルド | アルマグラディオン / バスティオンレックス | ファングレイジ | ヴォルカリオン / ラグナファング |
| 俊足 | ヴェロクス | ゼファリオン / クロスゲイル | スカイラッシュ | ボルトレイヴ / ストームアーク |
| 蓄積 | バルクドーム | タイタンバルワーク / グラビドーザー | エンバーポット | フレアグルム / マグナキルン |
| 静養 | ドリフトマーク | ネブラシェイド / ミストレイス | ムーンモス | ルナヴェルデ / エルダーモス |
| 過活動 | イグニダッシュ | ソルブレイザー / プロミネンスギア | クラックラン | レヴナントギア / フェニクレスト |

ロスターに記載済みの性別ヒント(アステリオン=オス系統、ミラフローラ/ルナヴェルデ/フェニクレスト=メス系統など)と完全に一致する。

## 分岐条件の実装

- 系統(第1週): `EvolutionEngine.chooseFamily`
- 職業(人型・第2週): `EvolutionEngine.chooseJob`
- 人型第4週: `choosesHumanoidFinal`(第3週の継続性50以上かつ機嫌60以上または絆40以上)
- 動物第4週: `chooseBeastFinal`(旧進化表の成熟体別条件を踏襲)
- ルート適性: `RouteAssigner`(孵化時抽選、移行時は決定的)

全71体の到達可能性は `BeastRouteTest.all 71 forms are reachable across routes and sexes` で常時検証している。
