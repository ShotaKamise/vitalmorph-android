## 女性人型の外見方針

- 全員を明確な成人型として描く
- 成人女性らしい頭身、表情、体格
- ボディライン、肩、腰、脚を活かした衣装
- 軽装鎧、フィットした衣装、ロングブーツなどでセクシーさを出す
- 戦闘能力と役割が一目で伝わるシルエット
- 露骨な裸や性的行為は扱わない
- 幼生、子供に見える形態、動物型は性的に扱わない

## 進化ルール

```text
生命卵
  -> 孵化時にオス・メスを決定
  -> 性別別の動物系幼生
  -> 食事と活動で動物系成長体3種へ分岐
  -> 第2週の傾向で人型7職業へ分岐
  -> 第3週の生活、機嫌、絆で上位人型または動物系最終形態へ分岐
```

同じ職業の上位形態へ進化してもよく、外見が大きく異なる動物系最終形態へ進化してもよい。最終週まで候補を変更できることを維持する。

オスからメス、メスからオスのフォームへは進化しない。性別による能力補正は付けない。
